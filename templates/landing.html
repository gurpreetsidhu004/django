How it works

Go to Upload Section

Upload your ducumnet.

Enter Category.

Enter title for the document.

Tags extraction and Image assignment

Here one select tags extraction model.

Images are automatically fetch from Unsplash.

Uploaded document store in cloudinary.

All the data will be save in datbase.

Upload to Ghost

User can browse all posts in web app.

Here user can choose diffrent image for tag.

There is button for send this post to Ghost.

Post id and tags id will store in database to avoid duplicate entries.

Architecture of our model

Model make use of Keras and LSTM to give prediction. LSTM uses previous memory to predict tags from given documents.For semantic relation word embeding is used.

  • Time distributed memory cell model is used.
  • Fully connected of 150 Neurons.
  • Three Neurons are used with sooftmax layer to predict key phrases.
